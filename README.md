## 🚀 Proyecto: Curriculum (Symfony)

Aplicación Symfony para la gestión y visualización de currículums.

### Requisitos

- PHP 8.1+
- Composer
- Symfony CLI

### Instalación

```bash
cd curriculum
composer install
symfony server:start
```

La aplicación arrancará en `http://localhost:8000`.
